\subsection{Detailed Description}
\doxyref{Partial\-Char\-CDI\_\-example.c}{p.}{PartialCharCDI__example_8c} This example reconstructs some Partial diffraction data (Diffractive imaging using partially coherent X rays). This approach assumes a gaussian coherence function and can characterise, the coherence of the beam as it reconstructs the image. For more detail, see Applied Physics Letters 99 (2011) - 'Simultaneous sample and spatial coherence characterisation using diffractive imaging' [doi: 10.1063/1.3650265] The shrinkwrap algorithm is used to improve the reconstruction. A combination of HIO and the error-reduction algorithm are used.
