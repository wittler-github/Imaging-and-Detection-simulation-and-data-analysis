\subsection{Detailed Description}
{\em \doxyref{Partial\-CDI\_\-example.c}{p.}{PartialCDI__example_8c}\/} This example reconstructs some Partial diffraction data (Diffractive imaging using partially coherent X rays Whitehead, L W, Physical review letters, 2009, v 103, 24, 243902). The shrinkwrap algorithm is used to improve the reconstruction. A combination of HIO and the error-reduction algorithm are used.
